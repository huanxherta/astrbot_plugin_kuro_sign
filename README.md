# 库街区自动签到 AstrBot 插件

## 功能
- 🎮 鸣潮/战双游戏签到
- 📝 论坛每日任务（签到、浏览、点赞、分享）
- 💰 金币查询
- 🔔 签到结果推送

## 命令
| 命令 | 说明 |
|------|------|
| `/库街区绑定 <token>` | 绑定你的库街区 token |
| `/库街区签到` | 执行签到（游戏+论坛） |
| `/库街区状态` | 查看绑定状态 |
| `/库街区解绑` | 解绑 token |

## 获取 Token
1. 浏览器登录 [库街区](https://www.kurobbs.com)
2. F12 → Network → 随便点一个请求
3. 找到 Cookie 中的 `user_token` 值
4. 发送 `/库街区绑定 <token>`

## 注意
- 仅供学习交流使用
- Token 保存在 AstrBot 本地，不会上传
